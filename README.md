this is a simple live chat room aplication 
## Documentation
* requirements
  
node.js

npm

**Clone and Install Script**

## install
```shell script
git clone https://github.com/Mr-joep/live-chat-room.git
cd live-chat-room
install npm
npm init -y
npm install express socket.io
```
## to start the chat room server 
```shell script
node server.js
```
